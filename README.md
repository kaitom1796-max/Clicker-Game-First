クリッカーゲーム（Unity制作）
■ 概要
Unityを使用した、クリックによるリソース収集とショップでの強化サイクルを楽しむクリッカーゲームです。ゲーム制作の基本であるUI制御、データ管理、ビルドまでの一連の流れを習得するために制作しました。

■ 技術スタック
開発環境: Unity 6

言語: C#

プラットフォーム: Windows (PC)

■ 実装済みの機能
コアロジック:

クリックによるスコア・コインの加算システム。

自動収入（アイドルゲイン）システムの実装。

ショップシステム:

コインを消費したクリック効率の強化。

特定の能力開放（各強化項目は1回購入制）。

セーブ機能:

PlayerPrefsを用いたデータの永続化。

ゲーム終了時の自動保存および起動時のロード機能。

UI/UX:

Canvasを用いた画面遷移（タイトル・ショップ・メイン）。

■ 開発における工夫点
オブジェクト指向の意識: スコア管理やショップ機能など、役割ごとにクラスを分割し、保守性の高いコード記述を意識しました。

データの整合性: 強化アイテム購入時に所持金が足りない場合のバリデーション処理など、例外が起きないロジックを構築しました。

■ 実行方法
リポジトリ内の Build フォルダをダウンロードします。

Clicker Game.exe を実行することでゲームが起動します。
